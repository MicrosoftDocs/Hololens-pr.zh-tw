# <a name="getting-around-hololens-2"></a>瀏覽 HoloLens 2

準備好探索全像投影的世界嗎？

本指南提供下列內容的簡介：

- 與混合現實互動
- 使用您的手和聲音與 HoloLens 2 上的全像影像互動
- 流覽 Windows 10 HoloLens (Windows 全像) 

## <a name="discover-mixed-reality"></a>探索混合現實

在 HoloLens 上，您可以將數位世界與實體環境混合，使其看起來像是世界的一部分。 即使全像全像投影，您隨時都可以看到您的環境、自由移動，以及與人員和物件互動。 我們將此體驗稱為「混合式事實」。

全像攝影框架將您的眼睛最敏感的地方放在您的詳細資料，並讓您的鏡頭讓您的周邊視覺清晰明瞭。 使用空間音效，您可以藉由聆聽來指出全像，即使是在您後方的影像。 而且，因為 HoloLens 瞭解您的實體環境，所以您可以在實際物件（例如資料表和牆壁）上放置全像投影。

使用您的智慧型手機很像是 HoloLens。 您可以使用您的手觸控和操作全像 windows、功能表和按鈕。  

一旦您知道這些基本互動之後，HoloLens 將會是一個嵌入式管理單元。

> [!TIP]
> 如果您目前已有 HoloLens，**提示** 應用程式會提供常值實際操作教學課程，以便在 HoloLens 上進行手動互動。  
> 使用開始手勢移至 **[開始] 或 [** 移至開始]，然後選取 [**提示**]。

## <a name="the-hand-tracking-frame"></a>手追蹤框架

HoloLens 的感應器可看到您的兩個英尺。 當您使用手時，您必須將其保留在該框架內，否則 HoloLens 不會看到它們。 但是，當您四處移動時，框架會隨您移動。  

![顯示 HoloLens 手追蹤框架的影像](./images/hololens-2-gesture-frame.png)

## <a name="touch-holograms-near-you"></a>觸及您附近的全像影像

當您附近有全息圖時，請將您的手拉近，而白色環形應該會出現在索引手指的秘訣上。  這是 **觸控游標** ，可協助您以精確的方式觸控和與全像影像互動。 若要 **選取** 某個東西 **，只要使用** 觸控游標來點一下。 使用手指 **輕** 觸在內容的表面上 **移動** 內容，就像是使用觸控式螢幕一樣。

若要 **抓取** 離您附近的全像投影，請將您的 **thumb** 和 **索引手指** 放在全像投影上，然後按住。 若要開始進行，請釋放您的手指。 使用此 **抓取手勢** 來移動、調整大小和旋轉混合實境首頁中的3d 物件和應用程式視窗。

若要顯示操作功能表（例如您將在 [開始] 功能表中的應用程式磚上找到的 **內容功能表**），請 **按住** 像在觸控式螢幕上的操作功能表。

## <a name="use-hand-ray-for-holograms-out-of-reach"></a>使用手中的手光線

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE3ZOum]

當您的手中沒有任何全息片時， **觸控游標** 會自動隱藏，而 **光線** 將會從您手中的手中顯示。 手光線可讓您從距離與全像影像互動。

> [!TIP]
> 如果您發現手中有雜亂的雜亂，您可以藉由說「隱藏手中」來隱藏它們。 若要使其重新出現，請說「顯示手片片」。

### <a name="select-using-air-tap"></a>使用攻點選取

若要使用 **手形光線** 選取某個內容，請遵循下列步驟：

1. 使用來自您掌上的手光線將專案設為目標。 您不需要引發整個 arm，也可以讓您的彎管更低且舒適。
1. 將您的索引指向最接近上限的指標。
1. 若要執行點 **一下手勢，** 請將您的 thumb 和索引合在一起，然後快速釋出。

   ![點一下手勢動畫](./images/hololens-air-tap.gif)

### <a name="grab-using-air-tap-and-hold"></a>使用空中點碰並按住

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE3Wxnh]

若要使用 **手光** 來抓取全像影像或滾動應用程式視窗內容，請從 **點擊** 點開始，但將手指保持在一起，而不是釋放它們。

使用 **點擊點並按住** 以使用手動光線執行下列動作：

- **滾動**。 若要將應用程式視窗內容滾動，請按一下並按住內容，然後將您的手的光線上下移動或側邊移至側面。
- **抓取**。 若要取得應用程式視窗或全像影像，請將應用程式標題列或全像投影的目標設為您的光線，然後按下並按住。
- **開啟內容功能表**。 若要開啟內容功能表，請使用滑鼠點選並按住您的手光線。

## <a name="start-gesture"></a>開始手勢

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE3Wxng]

開始手勢會開啟 **[開始] 功能表**。  若要執行開始手勢，請向您的手中拿手。 您會看到 [ **開始] 圖示** 出現在您的內部手腕上。 使用您的手勢來點擊此圖示。  [開始] 功能表將會開啟 **您正在尋找的位置**。

> [!TIP]
>
> - 只要您的手在手中，就可以使用 [開始] 手勢。  您不需要在 [開始] 圖示上向下查看。 將您的肘線保持在低和舒適，並查看您希望 [開始] 功能表開啟的方向。
> - 如果 [開始] 功能表沒有在您想要的位置開啟，只要移動您的頭部以重新放置它即可。
> - 如果您在 [開始] 功能表上讀取較小的文字時發生問題，請在開啟後更接近它。
> - 如果您的手稍微偏離框架的側邊，即使看不到圖示，仍然可以藉由使用內部手腕來查看 [開始] 功能表。

![顯示開始圖示和開始手勢的影像](./images/hololens-2-start-gesture.png)

若要 **關閉**[開始] 功能表，請在 [開始] 功能表開啟時進行啟動手勢。  您也可以查看 [開始] 功能表，然後說「Close」。

## <a name="start-menu-mixed-reality-home-and-apps"></a>[開始] 功能表、混合實境首頁和應用程式

您會在[[開始] 功能表](holographic-home.md)中找到已安裝的應用程式，而且您可以在[Microsoft Store](holographic-store-apps.md)中找到 HoloLens 的其他應用程式。

開啟 **[開始] 功能表**，然後選取 [**設定** 應用程式] 磚。 應用程式視窗將會在您之前開啟。

設定是使用 2d **應用程式視窗** 的 HoloLens 應用程式範例。  這與電腦上的 Windows 應用程式非常類似。

現在您可以再次開啟 **[開始] 功能表**，然後選取 **提示** 應用程式圖格。 應用程式的 3D **應用程式啟動器** 將會出現在您的前方。 若要開啟應用程式，您必須選取啟動器上的 [ **播放** ] 按鈕。

提示是 **沉浸式應用程式** 的範例。 沉浸式應用程式會在執行時讓您離開混合實境首頁，並成為您所看到的唯一應用程式。  若要結束，您需要啟動 [開始] 功能表，然後選取底部的 [**混合實境首頁**] 按鈕。

## <a name="move-resize-and-rotate-holograms"></a>移動、調整大小和旋轉全像影像

在混合實境首頁您可以使用手、手中的光線和語音命令來移動、調整大小和旋轉應用程式視窗和3D 物件。

### <a name="moving-holograms"></a>移動全息影像

遵循下列步驟來移動全息圖或應用程式：

1. 藉由捏合您的食指和手中的拇指，或面對您的手中，然後關閉第一個，來抓取全像影像。  在其藍色周框方塊內的任何位置抓取3D 全息圖。  針對應用程式視窗，抓取其標題列。
1. 如果不想要，請移動您的手以定位全像。 以這種方式移動應用程式視窗時，應用程式視窗會在移動時自動變成臉部，讓您更容易在新的位置使用。  
1. 釋放您的手指來放置。

### <a name="resizing-holograms"></a>調整全像大小

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE3ZYIb]

抓取並使用出現在3D 全像應用程式視窗角落的重 **設大小控點** ，以調整其大小。

針對應用程式視窗，當您以這種方式調整視窗內容的大小時，就會以更容易閱讀的方式來進行調整。

如果您想要調整應用程式視窗的大小，以便在視窗中顯示 **更多內容** ，請使用位於應用程式視窗側邊和下邊緣的調整大小控點。

有兩種方式可以調整離您更遠的全像影像。 您可以抓取全像影像的兩個角落，或使用調整大小控制項。

### <a name="rotating-holograms"></a>旋轉全像影像

若為3D 全像投影，請抓取並使用出現在周框方塊垂直邊緣的旋轉控點。

針對應用程式視窗，移動應用程式視窗會讓它自動旋轉並面對您。

您也可以一次抓取具有 **手** (或手光線) 的3d 全息圖或應用程式視窗，然後：

- 將您的手放在更接近的上方或更近，以調整全像影像大小。
- 將您的手移近和離您的身體，以旋轉全像投影。

### <a name="follow-me-stop-following"></a>接下來，停止

全像投影和應用程式視窗會保持您將它們放在世界各地。  如果您需要四處移動，而且想要讓應用程式保持可見，就不一定方便。 若要要求應用程式啟動或停止之後，請選取 [**關閉**] 按鈕旁邊的 [在應用程式視窗的右上角 **追蹤我** (]) 。  接下來的應用程式視窗，您也會將您帶到一個沉浸式應用程式外。
