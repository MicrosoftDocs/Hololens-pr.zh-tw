<span data-ttu-id="b79a6-108">HoloLens 上的 [開始] 功能表是您開啟應用程式、查看重要狀態資訊，以及存取相機等工具的位置。</span><span class="sxs-lookup"><span data-stu-id="b79a6-108">The Start menu on HoloLens is where you'll open apps, see important status info, and access tools like the camera.</span></span>

> [!TIP]
> <span data-ttu-id="b79a6-113">當 [開始] 功能表開啟時，可使用 [開始] 手勢將它關閉，或是看著 [開始] 功能表然後說「關閉」。</span><span class="sxs-lookup"><span data-stu-id="b79a6-113">When the Start menu is open, use the Start gesture to close it, or look at the Start menu and say "Close".</span></span>

### <span data-ttu-id="b79a6-118">在 [開始] 功能表上尋找應用程式</span><span class="sxs-lookup"><span data-stu-id="b79a6-118">Find apps on Start menu</span></span>

<span data-ttu-id="b79a6-124">在這兩個應用程式清單上，使用 [開始] 功能表右側的 [上一頁]\*\*\*\* 和 [下一頁]\*\*\*\* 按鈕，可分頁瀏覽清單中的所有應用程式。</span><span class="sxs-lookup"><span data-stu-id="b79a6-124">On both app lists, use the **Page up** and **Page down** buttons on the right side of the Start menu to page through all the apps in the list.</span></span>  <span data-ttu-id="b79a6-125">這兩個應用程式清單會自動開啟到裝置工作階段期間上次使用的頁面。</span><span class="sxs-lookup"><span data-stu-id="b79a6-125">Both app lists will automatically open to the page that was last used during a device session.</span></span>

### <span data-ttu-id="b79a6-128">從 [開始] 功能表開啟應用程式</span><span class="sxs-lookup"><span data-stu-id="b79a6-128">Open apps from Start menu</span></span>

## <span data-ttu-id="b79a6-143">在 HoloLens 上使用應用程式</span><span class="sxs-lookup"><span data-stu-id="b79a6-143">Using apps on HoloLens</span></span>
