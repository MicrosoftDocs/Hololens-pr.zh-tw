### <span data-ttu-id="e3496-152">自動目視位置支援</span><span class="sxs-lookup"><span data-stu-id="e3496-152">Auto Eye Position Support</span></span>

<span data-ttu-id="e3496-153">在 HoloLens 2 中，目視位置可讓您實現正確的全息圖定位、舒適的觀賞體驗，以及改善顯示品質。</span><span class="sxs-lookup"><span data-stu-id="e3496-153">In HoloLens 2, eye positions enable accurate hologram positioning, comfortable viewing experience and improved display quality.</span></span> <span data-ttu-id="e3496-154">眼睛位置會計算為眼睛追蹤結果的一部分。</span><span class="sxs-lookup"><span data-stu-id="e3496-154">Eye positions are computed as part of the eye tracking result.</span></span> <span data-ttu-id="e3496-155">不過，這需要每個使用者都能透過目視追蹤校準，即使在體驗不需要目視眼睛的輸入時也一樣。</span><span class="sxs-lookup"><span data-stu-id="e3496-155">However, this requires each user to go through eye tracking calibration, even when the experience does not require eye gaze input.</span></span>

<span data-ttu-id="e3496-156">\*\*自動眼睛位置 (AEP) \*\*啟用這些案例，並使用互動式方式來計算使用者的目視位置。</span><span class="sxs-lookup"><span data-stu-id="e3496-156">**Auto Eye Position (AEP)** enables these scenarios with an interaction-free way to compute eye positions for the user.</span></span>  <span data-ttu-id="e3496-157">自動目視位置會在使用者將裝置放在背景的時刻自動開始運作。</span><span class="sxs-lookup"><span data-stu-id="e3496-157">Auto Eye Position starts working in the background automatically from the moment the user puts the device on.</span></span> <span data-ttu-id="e3496-158">如果使用者沒有先前的目視追蹤校準，自動目視位置將會在較小的處理時間之後，在顯示系統中開始提供使用者的目視位置。</span><span class="sxs-lookup"><span data-stu-id="e3496-158">If the user does not have a prior eye tracking calibration, Auto Eye position will start providing the user's eye positions to the display system after a small processing time.</span></span> <span data-ttu-id="e3496-159">這個處理時間通常介於 20-60 秒之間。</span><span class="sxs-lookup"><span data-stu-id="e3496-159">This processing time typically is between 20 - 60 seconds.</span></span> <span data-ttu-id="e3496-160">使用者資料不會保留在裝置上，因此，如果使用者要卸載並將裝置重新開機或從睡眠狀態喚醒，就會重複此處理程式。</span><span class="sxs-lookup"><span data-stu-id="e3496-160">The user data is not persisted on the device and hence this process is repeated if the user takes off and puts the device back on or if the device reboots or wakes up from sleep.</span></span>
