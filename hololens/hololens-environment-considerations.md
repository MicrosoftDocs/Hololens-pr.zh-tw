# <a name="hololens-environment-considerations"></a>HoloLens 環境考慮

HoloLens 將全像是「真實」世界，將全像放在您的環境中。 在牆上的全像的應用程式視窗會「停止回應」，這是桌面上的全像 ballerina 旋轉，小兔子耳在不知情 friend 的標頭上。 當您使用沉浸式遊戲或應用程式時，全像世界會散佈以填滿您的環境，但您仍然可以查看並四處移動空間。

即使您關閉裝置，您所放置的全息照片仍會保留在您放置它們的位置。

## <a name="setting-up-an-environment"></a>設定環境

HoloLens 裝置知道如何藉由在空間中 *追蹤* 使用者，來放置穩定且準確的全像全像。 如果沒有適當的追蹤，裝置就不會瞭解環境或其中的使用者。 全像投影可以出現在錯誤的位置，不會每次出現在相同的位置，或根本不會出現。 用來追蹤使用者的資料會以 *空間對應* 表示。  

追蹤效能會受到使用者所在環境的高度影響，並調整環境以引發穩定且一致的追蹤，而不是科學。 許多不同的環境因素都融合在一起以啟用追蹤，但作為混合現實開發人員，您可以牢記幾個因素來微調空間，以進行更好的追蹤。

### <a name="lighting"></a>光源

Windows Mixed Reality 使用視覺效果燈光來追蹤使用者的位置。 當環境太鮮時，相機可能會變得飽和，而不會看到任何內容。 如果環境太暗，攝影機就無法挑選足夠的資訊，也不會看到任何內容。 光源應該是甚至是很好的，人類可以看到，而不會有太大的好處，但不太好，光是很難查看。  

整體維度區域中有點亮亮的區域也會有問題，因為攝影機必須在移入和移出亮空間時進行調整。 這可能會導致裝置「遺失」，並認為光線的變更等於位置的變更。 區域中穩定的光線層級將會導致更好的追蹤。  

任何戶外照明也會在追蹤器中造成不穩定的情況，因為 sun 在一段時間內可能會有顯著的差異。 例如，在夏天和冬季的相同空間中進行追蹤，可能會產生明顯不同的結果，因為外部的第二個光線可能會在一年的不同時間更高。  

如果您有 luxmeter，穩定的 500-1000 lux 是不錯的開端。  

#### <a name="types-of-lighting"></a>光源類型

空間中不同類型的燈光也會影響追蹤。 透過電源執行的 AC 電力來輕燈泡脈衝-如果 AC 頻率為 50 Hz，則在 50 Hz 的燈跳動。 對於人類來說，並不會注意到這個閃爍。 不過，HoloLens 的 30 fps 攝影機會看到這些變更，有些畫面格會很亮，有些則會不佳，有些則會因為攝影機嘗試補償淺脈衝而過度公開。  

在美國，電力頻率標準為 60 Hz，因此燈泡脈衝會與 HoloLens 的畫面播放速率-60 Hz 脈衝對齊 HoloLens ' 30 FPS 畫面播放速率。 不過，許多國家/地區的 AC 頻率標準為 50 Hz，這表示某些 HoloLens 的畫面格會在脈衝期間進行，其他則不會發生。 特別是，歐洲的螢光燈已知會造成問題。  

有幾件事您可以嘗試解決閃爍的問題。 溫度、燈泡年齡和準備週期是螢光燈閃爍的常見原因，並取代燈泡可能會有説明。 燈泡並確定目前的繪圖是常數，也可以提供協助。  

### <a name="items-in-a-space"></a>空間中的專案

HoloLens 使用獨特的環境地標，也稱為 *功能*，在空間中找出本身。  

裝置幾乎不會在功能不佳的區域中進行追蹤，因為裝置無法得知空間的所在位置。 將功能加入至空間的牆通常是改善追蹤的好方法。 海報、標貼至牆、工廠、唯一物件或其他類似專案的符號都有説明。 很多人都是一個很好的環境範例，會導致良好的追蹤-單一區域中有許多不同的功能。  

此外，請在相同的空間中使用唯一的功能。 例如，相同的海報在牆上重複多次，會導致裝置混淆，因為 HoloLens 不知道其所查看的重複海報。 新增獨特功能的其中一種常見方式是使用遮罩磁帶的線，在空間的牆壁和樓層之間建立獨特的非重複模式。  

問您自己的一個好問題是：如果您只看到少量的場景，可以在空間中唯一找出自己嗎？ 如果沒有，則裝置可能也會發生追蹤問題。

#### <a name="wormholes"></a>蟲 洞

如果您有兩個看起來相同的區域或區域，則追蹤器可能會認為它們是相同的。 如此一來，裝置就會將自身引誘成在其他地方。 我們稱之為這類重複區域 *wormholes*。  

若要避免 wormholes，請嘗試避免相同空間中的相同區域。 相同的區域有時可能包含原廠站、建築物、伺服器機架或工作工作站上的 windows。 標記區域或將獨特功能新增至每個類似的區域，有助於減輕 wormholes。

### <a name="movement-in-a-space"></a>在空間中移動

如果您的環境不斷地改變和變更，則裝置不會針對找出穩定的功能。  

空間中移動的物件越多，包括人們，就越容易遺失追蹤。 移動傳送帶、不同結構狀態中的專案，以及空間中的許多人都已知會造成追蹤問題。

HoloLens 可以快速適應這些變更，但只有在該區域清楚地顯示裝置時才會顯示。 通常不會經常出現的區域可能會在實際情況下落後，這可能會導致空間對應發生錯誤。 例如，使用者會掃描朋友，然後在 friend 離開房間的時候解決問題。 在使用者重新掃描現在空白空間之前，friend 的「准刪除」表示會保存在空間對應資料中。

### <a name="proximity-of-the-user-to-items-in-the-space"></a>使用者與空間中專案的相近程度

同樣地，在靠近眼睛的物件上，HoloLens 掙扎的物件是否接近其攝影機。 如果物件太接近了兩張相機，或是某個物件封鎖了一個相機，則裝置會有更多的追蹤物件問題。  

攝影機從物件看得不到接近 15 cm。

### <a name="surfaces-in-a-space"></a>空間中的表面

強烈的反射表面可能會根據角度而有所不同，這會影響追蹤。 您可以想像全新的汽車-當您四處移動時，光線會反映，而您在移動時看到不同的物件。 在追蹤器中，表面上反映的不同物件代表變動的環境，而裝置會失去追蹤。

較不亮的物件比較容易追蹤。

### <a name="wi-fi-fingerprint-considerations"></a>Wi-Fi 指紋考慮

只要已啟用 Wi-Fi，地圖資料將會與 Wi-Fi 指紋相互關聯，即使未連接到實際的 WiFi 網路/路由器也一樣。 若沒有 Wi-Fi 資訊，空間和全息空間可能會稍微慢于辨識。 如果 Wi-Fi 的信號會大幅變更，則裝置可能會認為它是在不同的空間中。

網路識別 (例如 SSID 或 MAC 位址) 不會傳送到 Microsoft，而且所有 Wi-Fi 參考都會保留在本機 HoloLens 上。

## <a name="mapping-new-spaces"></a>對應新的空格

當您輸入新的空間 (或載入現有的空間) 時，您會看到在空間上分配的網格圖形。 這表示您的裝置會對應您的環境。 雖然 HoloLens 會在一段時間內學習空間，但是有一些秘訣和訣竅可以對應空格。

## <a name="environment-management"></a>環境管理

有兩個設定可讓使用者「清除」全像「清理」，並導致 HoloLens 「忘記」空格。 它們存在於 [設定] 應用程式的 **全像投影和環境** 中，第二項設定也會出現在 [設定] 應用程式的 [**隱私權**] 下。  

1. **刪除附近** 的全像投影。 當您選取此設定時，HoloLens 將會清除裝置所在之「目前空間」的所有錨定的全像投影和所有儲存的地圖資料。 一旦重新建立映射之後，就會為該位置建立新的地圖區段，並將其儲存在該位置的資料庫中。

1. **刪除所有的全像影像**。 藉由選取此設定，HoloLens 將會清除整個空間資料庫中的所有地圖資料和錨定的全像投影。 將不會重新探索任何全息表，而且必須將所有的全像全像放在資料庫中，將地圖區段最新地儲存。

## <a name="hologram-quality"></a>全像影像品質

您可以將全像投影放在您的環境中（高度、低），但您會看到這些都是在您眼睛前方的全像攝影[框架](/windows/mixed-reality/holographic-frame)。 若要取得最佳的觀點，請務必調整您的裝置，讓您可以看到整個畫面格。 您也不需要在您的環境中四處探索！

[為了讓您的](/windows/mixed-reality/hologram)全像簡潔、清楚且穩定，您的 HoloLens 必須為您校正。 當您第一次設定 HoloLens 時，將會引導您完成此程式。 之後，如果全像投影看起來不正確，或者您看到許多錯誤，您可以進行調整。

如果您遇到問題-對應空間，請嘗試刪除附近的全像投影，然後重新對應空間。

### <a name="calibration"></a>校正

如果您的全像是抖動或晃動，或是您在放置全像影像時遇到問題，首先要試試的就是 [校正應用程式](hololens-calibration.md)。 如果您在使用 HoloLens 時遇到任何不適感，此應用程式也會有所説明。

若要取得校正應用程式，請移至 **設定**  >  **系統**  >  **公用程式**。 選取 [ **開啟校正** ]，然後依照指示進行。

如果有其他人要使用您的 HoloLens，則應該先執行校正應用程式，以正確地設定裝置。

請參閱[材料和物質](https://www.microsoft.com/legal/compliance/materials-substances)中的「HoloLens」詳細資料，> 有關環境合規性 (PDF) 的文章33。

以下是使用裝置時應遵循的一些指導方針：

1. 將裝置儲存在溫度範圍內的環境內 (在待命或關閉) 一小時後，再使用裝置。
1. 在溫度範圍內的環境中使用裝置。
1. 使用裝置室內。
1. 以陰影使用裝置;即使是在 windows 或 skylights 上，也能避免直接的陽光。
1. 如果您遵循上述指導方針但遇到非預期的熱問題，請遵循下列步驟來提交 [意見](hololens-feedback.md)反應。 
    1. 確定已在您的裝置上啟用 **完整** 或 **選擇性** 的遙測。 如果未啟用，請加以啟用。 
    >[!CAUTION]
    > 遙測並非針對熱事件追溯-必須在熱事件期間啟用，否則將不會捕捉必要的資料。
    
    2. 重現加熱問題。
    3. 包含發生過熱的日期和時間。
    4. 提交 [意見](hololens-feedback.md)反應。
